==Description==

A plugin to show the live IPL cricket score in your wordpress blog using shortcode and widget. Use the plugin configuration page (Settings -> Kuvilam IPL Livescore) to customize the widget colors, label and score refresh timing.


==Installation==

1. Upload the plugin files to the '/wp-content/plugins' directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Kuvilam IPL Livescore screen to configure the plugin

==Technical Details==
The cricket live score data is provided by (https://www.iplcricketmatch.com/) and the service API URLs are given below. Please check this url (https://www.iplcricketmatch.com/privacy-policy/) for the privacy and policy details.

 - http://pub.wpdemo.biz/ils.json -  for the initial score card
 - http://pub.wpdemo.biz/ilsmin.json - for the ajax refresh
